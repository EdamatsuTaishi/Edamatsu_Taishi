# 第5回レポート：モース分解

## 1. テーマ

本レポートでは、1次元力学系のモース分解を可視化する。

対象とする力学系は

\[
\dot{x}=x-x^3
\]

である。

平衡点は、\(\dot{x}=0\) を満たす点なので、

\[
x-x^3=0
\]

より、

\[
x=-1,\quad x=0,\quad x=1
\]

である。

また、ベクトル場

\[
F(x)=x-x^3
\]

を微分すると、

\[
F'(x)=1-3x^2
\]

となる。

各平衡点では、

\[
F'(-1)=F'(1)=-2,\qquad F'(0)=1
\]

である。

したがって、\(x=-1\) と \(x=1\) は安定平衡点であり、\(x=0\) は不安定平衡点である。

## 2. 大域アトラクタ

この力学系の大域アトラクタは、

\[
\mathcal{A}=[-1,1]
\]

である。

この大域アトラクタには、3つの平衡点と、それらを結ぶ軌道が含まれている。

\(x=0\) から出発した軌道は、わずかな初期値の違いによって、\(x=-1\) または \(x=1\) に向かう。

## 3. モース分解

この力学系のモース分解の一例は、

\[
M_-=\{-1\},\qquad
M_0=\{0\},\qquad
M_+=\{1\}
\]

である。

ここで、

- \(M_-\) は安定平衡点 \(x=-1\) に対応する。
- \(M_0\) は不安定平衡点 \(x=0\) に対応する。
- \(M_+\) は安定平衡点 \(x=1\) に対応する。

完全軌道による有向接続は、

\[
M_0\rightarrow M_-
\]

および、

\[
M_0\rightarrow M_+
\]

である。

これは、不安定なモース集合 \(M_0\) から、2つの異なる安定なモース集合へ移動できることを表している。

## 5. モース分解との関係

モース分解は、大域アトラクタの内部に存在する不変集合と、それらの間を結ぶ軌道を整理する方法である。

この例では、3つの平衡点をそれぞれモース集合として扱っている。

モースグラフでは、モース集合が頂点となり、モース集合を結ぶ完全軌道が有向辺となる。

そのため、この力学系のモースグラフは、

\[
M_0\rightarrow M_-
\]

および、

\[
M_0\rightarrow M_+
\]

という2つの有向辺を持つ。

このグラフは、安定な平衡点を列挙するだけではなく、どの状態からどの状態へ移動できるかという、可能な遷移の構造も記録している。

## 6. 統合情報との関係

講義資料では、モース集合を情報構造のノード、モース集合を結ぶ完全軌道を有向辺として扱う。

この例では、現在の状態が不安定なモース集合 \(M_0\) にあるとき、将来は安定なモース集合 \(M_-\) または \(M_+\) に向かう。

そのため、この図は、現在の状態が可能な未来をどのように制約しているかを表している。

モース集合間の接続に強さを与え、それを正規化することで、将来の遷移確率を定義できる。

この遷移確率から、現在の状態が可能な未来について持つ情報である結果レパートリーを構成できる。

同様に、過去方向の接続を用いることで、現在の状態が可能な過去について持つ情報である原因レパートリーを構成できる。

さらに、力学系を2つの部分に分割し、分割前と分割後の原因レパートリーおよび結果レパートリーを比較することで、統合情報 \(\phi\) を考えることができる。

この図では統合情報 \(\phi\) の数値計算までは行っていない。

しかし、統合情報を定義するための基礎となる連続時間の原因・結果構造を、モース集合と有向接続によって表している。
